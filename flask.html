O que é o Flask?

Flask é um micro framework que utiliza a linguagem Python para criar aplicativos Web. Não se deixe enganar pelo prefixo “micro”, em algumas extensões o Flask é poderoso e ideal para quem busca: Simplicidade; Rapidez; Soluções para projetos pequenos; Aplicações robustas. Lançado em 2010 e desenvolvido por Armin Ronacher, a estrutura do Flask permite desenvolver aplicativos da web facilmente.Ronacher ganhou fama por liderar uma comunidade de entusiastas do Python chamada Poocco. O Flask é baseado no kit de ferramentas Werkzeg WSGI e na biblioteca Jinja2. Ambos são projetos herdados da Poocco.

Mas afinal, o que é um Micro-framework?

Um Micro-Framework são Frameworks modularizados que possuem uma estrutura inicial muito mais simples quando comparado a um Framework convencional. Podemos dizer que o micro-framework é uma versão minimalista destes frameworks, sendo bastante utilizado para criação de microsserviços, como APIs RESTful. No artigo anterior, fizemos uma comparação de objetos do mundo real para exemplificar o funcionamento de um micro-framework: Pense em um Micro-Framework como uma peça de lego. Inicialmente, um projeto criado com o micro-framework possui apenas o básico para funcionar, (normalmente, sistema de rotas), porém, ao decorrer do projeto, podem haver necessidades para utilização de outros recursos como, conexão de banco de dados, sistemas de templates, envio de email, etc. A partir desta necessidade, novas bibliotecas são “encaixadas” no projeto, como uma estrutura de lego.
